## Convenciones de nomenclatura
|  			Sigla 		                    |  			Significado 		                                                                                         |
|----------------------------|-------------------------------------------------------------------------------------------------------|
|  			&lt;ext> 		                 |  			Nombre que identifica el formato 			del archivo o el tipo de datos que contiene. 		                       |
|  			&lt;nro_TP> 		              |  			Número entero que identifica al 			Trabajo Práctico 		                                                    |
|  			&lt;TipoDeInv> 		           |  			Identifica el tipo de investigación 			del trabajo, pudiendo ser poster o charla. 		                      |
|  			&lt;TemaDeLaClase> 		       |  			Nombre que va a 			describir el tema que se llevara a cabo en clases, como el de 			requerimientos agiles 		 |
|  			&lt;NroClase> 		            |  			Número único 			secuencial que nos permite ordenar de manera cronológica las 			secciones de clase. 		       |
|  			&lt;nro_Sprint> 		          |  			Número único 			secuencial que permite identificar a los sprints.  			 		                                    |
|  			&lt;Resultado> 		           |  			Nombre 			representativo del incremento obtenido en cada sprint. 		                                       |
|  			&lt;NroSemana> 		           |  			Número único 			secuencial que nos permite ordenar de manera cronológica las 			semanas. 		                  |
|  			&lt;Descripcion> 		         |  			Nombre único 			representativo que especifica el contenido del archivo. 		                                |
|  			&lt;Nombre_Bibliografía> 	 |  			Identificador único y 			estandarizado del archivo para determinar su contenido. 		                       | 

## Criterio de línea base
Se establecerá una línea base cada vez que se obtenga una retroalimentación satisfactoria (aprobación), por parte del cliente, sobre las entregas realizadas de los trabajos prácticos evaluables.
